J'ai donc commencé la creation d'un projet de site web pour un POMODORO.
J'ai eu cette idée suite a un stream de kevintran sur twitch qui stream lorsqu'il travaille , et utilise le POMODORO pour travailler et motivé les autres.





QU'EST-CE QUE LE POMODORO
C'est une technique de travail, qui consiste a  travailler pendant 25min puis faire une pause de 5min et ainsi de suite.




QUE CONTIENT CE SITE
J'ai mis un fond video (gif) lofi chill qui tourne en boucle et qui pemet de mettre l'utlisateur dans une ambiance de travaille.



QUE ME RESTE T IL A FAIRE
- Ajouter le timer de 25min, qui declanchera une alarme a la fin, puis un bouton d'arrete de cette alarme qui declanchera le commencement du timer de 5min et ainsi 
de suite.
- En dessous du timer, un bouton stop pour arreté le timer
- Un bouton reset, pour recommencé la session (25min)




 QU'EST CE QUE J'AIMERAI AJOUTE
 Le timer sera centré, avec un cercle autour qui disparé petit a petit avec le timer
 
 j'aimerai ajouté un compteur pour connaitre combien de session l'utilisateur a t il effectué
 (session = 25min)
 
 Que l'utilisateur puisse avoir accès a une playlist, où lui sera proposé plusieurs musique chill, s'il souhaite travailler avec une musique en fond.
